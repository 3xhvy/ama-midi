CSS Variable Strategy — Two Token Sets

/* Semantic tokens that flip with theme */
:root {
  --shell-bg: #F8F7FF;         /* panels, topbar, bottombar */
  --shell-surface: #FFFFFF;
  --shell-border: #E8E6F0;
  --shell-text: #1A1635;
  --shell-muted: #6B6585;

  --canvas-bg: #F0EEF8;        /* editor grid area */
  --canvas-surface: #E8E6F0;
  --canvas-border: #D4D0E4;
  --canvas-text: #1A1635;
  --canvas-grid: rgba(0,0,0,0.06);
  --canvas-grid-bold: rgba(0,0,0,0.12);
}

.dark {
  --shell-bg: #13111E;
  --shell-surface: #1E1B2E;
  --shell-border: #2D2847;
  --shell-text: #E2DEFF;
  --shell-muted: #6B6585;

  --canvas-bg: #0F0D1A;
  --canvas-surface: #1A1729;
  --canvas-border: #2D2847;
  --canvas-text: #E2DEFF;
  --canvas-grid: rgba(255,255,255,0.06);
  --canvas-grid-bold: rgba(255,255,255,0.12);
}

/* Brand colors stay constant regardless of theme */
--color-primary: #6C63FF;      /* always */
--color-success: #10B981;
--color-warning: #F59E0B;
--color-error: #EF4444;
Key insight: Split tokens into --shell-* (panels/bars) and --canvas-* (grid area). Both flip together via .dark class on <html>. Note colors and brand colors are theme-independent — they always pop against any background.

Responsive Breakpoints

Desktop (≥1280px)    Full 5-zone: Left 240px + Center + Right 280px
Tablet (768–1279px)  Left collapses to icons-only (48px) or hidden
                     Right becomes slide-over overlay (280px)
                     TopBar compact: hide center buttons, use overflow menu
Mobile (< 768px)    Single column: Canvas fills viewport
                     TopBar → minimal (song name + menu hamburger)
                     Left/Right → full-screen sheets (slide from edge)
                     BottomBar → sticky with essential controls only

Grid area contrast (light vs dark)

Important: Note circles (preset colors like #6C63FF, #3B82F6, #10B981) are high-contrast against both light and dark canvas backgrounds. The grid line tokens (--canvas-grid) auto-flip between rgba(0,0,0,0.06) (light) and rgba(255,255,255,0.06) (dark). No note color changes needed per theme.
