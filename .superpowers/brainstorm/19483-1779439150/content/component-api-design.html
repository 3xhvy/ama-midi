Section 2: Component API Design

Variant-driven primitives with consistent prop patterns

Shared Pattern: Every component follows this contract

// Every UI component:
1. Accepts className for override/extension
2. Forwards ref (React.forwardRef)
3. Spreads remaining HTML props (...rest)
4. Uses cn() helper for conditional class merging
5. Uses design tokens (never hardcoded hex values)

Button

Props
variant: 'primary' | 'secondary' | 'ghost' | 'danger'
size: 'sm' | 'md' | 'lg'
rounded: boolean (pill shape)
loading: boolean
disabled: boolean
icon: ReactNode (left icon slot)
Design.md Mapping
▶ Preview = primary, rounded
✓ Save = secondary, rounded
Undo = ghost, sm
Delete = danger

Badge / StatusBadge

Badge Props
variant: 'default' | 'success' | 'warning' | 'error' | 'muted'
size: 'sm' | 'md'
icon: ReactNode
StatusBadge Props
status: 'synced' | 'needsReview' | 'outdated' | 'draft'
// Auto-maps to colors from STATUS_COLORS
// Auto-maps to icon (✓, ⚠, ✕, ○)
✓ Synced ⚠ Needs Review ✕ Outdated

Avatar / AvatarStack

Avatar Props
src?: string
name: string (fallback → initials)
size: 'xs' | 'sm' | 'md' | 'lg' (24/32/40/48px)
showOnline?: boolean (green dot)
AvatarStack Props
users: { name, avatarUrl?, id }[]
max: number (default 5)
size: 'sm' | 'md'
// Shows +N overflow pill automatically

Modal (Radix Dialog)

Modal.Root — wraps Radix Dialog.Root
Modal.Trigger — opens the modal
Modal.Content — centered card (400px, rounded-xl, shadow-lg)
Modal.Header — title + close button
Modal.Body — content area
Modal.Footer — action buttons row

// Backdrop: rgba(26,22,53,0.6) per Design.md
// Escape key closes, focus trapped, body scroll locked

Tabs (Radix Tabs)

Tabs.Root — wraps Radix Tabs.Root
Tabs.List — horizontal tab bar with bottom border
Tabs.Trigger — tab button (active = purple underline)
Tabs.Content — panel content

// Used in: RightPanel (Details/Validation/History)
// Keyboard: Tab key switches per Design.md shortcuts

ToggleGroup (Radix ToggleGroup)

variant: 'default' | 'editor' (light vs dark bg)
items: { value, label }[]
value: string
onValueChange: (value) => void

// Used for: View mode (Composer/Developer/QA), Zoom (1x/2x/4x)
// Active item: bg-primary text-white
