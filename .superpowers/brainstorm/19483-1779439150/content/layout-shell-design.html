EditorShell Component API

interface EditorShellProps {
  topBar: ReactNode          // 64px fixed header
  leftPanel: ReactNode       // 240px, collapsible
  rightPanel: ReactNode      // 280px, collapsible
  bottomBar: ReactNode       // 48px fixed footer
  children: ReactNode        // center zone (PianoRoll)
  leftCollapsed?: boolean    // hide left panel
  rightCollapsed?: boolean   // hide right panel
}

// Usage in EditorPage:
<EditorShell
  topBar={<TopBarContent />}
  leftPanel={<LayerPanel />}
  rightPanel={<DetailsTabs />}
  bottomBar={<PlaybackBar />}
>
  <PianoRoll songId={songId} />
</EditorShell>

Key Layout Rules (from Design.md)

100vh, no outer scroll
Only the PianoRoll timeline scrolls internally. The shell fills the viewport exactly.
Light shell / Dark center
Panels = --color-surface (white)
Center = --color-editor-bg (#13111E)
Collapsible panels
Left/Right can collapse with animation (slide-in-right). Toggle via keyboard or button.
History panel overlay
Slides in from right (320px), overlays RightPanel. Uses panel-right animation class.
